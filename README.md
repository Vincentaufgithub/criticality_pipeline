# criticality_pipeline

Pipeline to load and sort spiking data, in order to run criticality analyses.
Relies on MR.Estimator, pynapple, as well as several packages from the Loren Frank lab.

So far, the pipeline is written for the following dataset:
https://datadryad.org/stash/dataset/doi:10.7272/Q61N7ZC3

We intend on extending the pipeline to facilitate working with different data formats and analysis goals.
